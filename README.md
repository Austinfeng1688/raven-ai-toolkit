## 什么是 AI 搭档？

AI 搭档不是聊天机器人。区别在这里：

| 聊天机器人 | AI 搭档 |
|-----------|---------|
| 聊完就忘 | 持久记忆，跨对话 |
| 等你问 | 主动提醒、预判需求 |
| 没有工具 | 读文件、搜索、执行 |
| 回答一个问题 | 规划多步、判断路径 |

这个仓库就是帮你把你的 ChatGPT 升级成你自己的 Raven。
