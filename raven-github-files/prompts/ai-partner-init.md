# AI 搭档初始化模板

> 把这个 Prompt 发给你用的 AI（ChatGPT / Claude / 任何大模型），让它开始了解你。

---

## 使用方法

1. 复制下面的 Prompt
2. 粘贴发给你的 AI
3. 认真回答它的问题（越详细越好）
4. 让它把结果总结成一份画像，保存下来
5. 以后每次新对话，先把画像发给它

---

## Prompt 模板

```
我想把你从"对话工具"升级为"AI 搭档"。
你要做的不是回答完就忘掉——而是记住我、理解我、预判我的需求。

现在请你像采访一样问我问题，覆盖以下维度，直到你对我有比较完整的理解：

1. 性格——我是主动的人还是被动的人？我容易生什么气？
2. 爱好——空闲时间在做什么？什么对我最重要？
3. 品味——我觉得什么东西是"好"的？什么东西我受不了？
4. 工作——我靠什么赚钱？我最骄傲的事是什么？
5. 恐惧——我最怕什么？我讨厌和什么样的人共事？
6. 欲望——五年后我想站在哪里？钱、名、自由、影响力，我排什么顺序？
7. 习惯——我的日常节奏是什么样的？

每个维度问 2-3 个问题。不要一次全问完——分批问，根据我的回答深入追问。

问完之后，把你对我的理解整理成一份"完整画像"，用 Markdown 格式呈现，包含：
- 核心特质
- 优势
- 盲区
- 行为准则（你能怎么帮我）

这份画像是以后我们所有对话的基础。每次新对话，我会先把画像发给你，你就能立刻进入状态。
```

---

## 进阶用法

如果你想要更强的一步——把这个 Prompt 的输出保存为一个文件（比如 `my-portrait.md`），然后：

- **用 ChatGPT 桌面版**：把文件拖进对话框
- **用 Claude Code**：放到工作目录下
- **用 API**：作为 System Prompt 传入

这样每个新对话窗口，你的 AI 都不会忘记你是谁。
